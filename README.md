# Curso de Diagramas de Flujo y Pseudocódigo

¡Bienvenido al repositorio oficial del curso! Este espacio está diseñado para almacenar todo el material educativo, presentaciones y soluciones de los ejercicios prácticos desarrollados a lo largo de las clases.

## 🎯 Objetivos del Curso
* Comprender la lógica de programación a través de herramientas visuales.
* Desarrollar algoritmos eficientes utilizando pseudocódigo.
* Dominar las estructuras de control secuenciales, condicionales y repetitivas.

## 📁 Estructura del Repositorio

El material se organiza por módulos para facilitar la navegación:

* **/Presentaciones**: Diapositivas y material teórico en formato PDF o PPTX.
* **/Ejercicios**: Códigos fuente y representaciones visuales de los algoritmos.
  * Archivos `.psc`: Código ejecutable en PSeInt.
  * Archivos `.png` / `.jpg`: Diagramas de flujo correspondientes.

## 🛠️ Herramientas Utilizadas
* **PSeInt**: Software para escribir y probar el pseudocódigo.
* **Lucidchart / Draw.io**: Herramientas para el diseño de diagramas de flujo.

## 📚 Contenido del Curso

### Módulo 1: Estructuras Secuenciales
* Declaración y asignación de variables.
* Entrada y salida de datos (Leer y Escribir).
* Operaciones matemáticas básicas.

### Módulo 2: Estructuras Condicionales
* Condicionales simples y dobles (Si-Entonces / Sino).
* Condicionales anidadas.
* Condicionales múltiples (Según / Caso).

### Módulo 3: Ciclos y Bucles
* Estructura repetitiva Mientras (While).
* Estructura repetitiva Repetir (Do-While).
* Estructura repetitiva Para (For).
